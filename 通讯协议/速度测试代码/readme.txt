先准备好所需的python环境以及serial包
然后启动之后输入一个名为“/dev/ttyACM0”的串口，等待它测试完成。
**********
我试了一下，
Windows的上传速度（单片机传给上位机，后面也是这个意思）为2.048KBytes/s,
下载速度（上位机传给下位机）为5588.99KBytes/s。
Ubuntu上传速度为1075.2KBytes/s,
下载速度为392.264KBytes/s。
树莓派上传速度为493.568KBytes/s,
下载速度为423.936KBytes/s。
**********
当然，上面的数字仅供参考，具体的传输速度与系统，连接等诸多因素有关，上述数字不代表最终传输速度。
